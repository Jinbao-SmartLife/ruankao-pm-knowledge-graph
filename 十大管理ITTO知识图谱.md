# 十大管理 ITTO 知识图谱

## 一、五大过程组 × 十大知识领域矩阵

> 颜色标记：🔵一次/预定义点 | 🟢定期开展 | 🔴整个期间开展 | 🟡整个期间反复开展

| 知识领域 | 启动过程组 | 规划过程组 | 执行过程组 | 监控过程组 | 收尾过程组 |
|---|---|---|---|---|---|
| **1.整合管理** | 🔵制定项目章程 | 🔵制定项目管理计划 | 🔴指导与管理项目工作<br>🔴管理项目知识 | 🔴监控项目工作<br>🔴整体变更控制 | 🔵结束项目或阶段 |
| **2.范围管理** | | 🔵规划范围管理<br>🔵收集需求<br>🟡定义范围<br>🔵创建WBS | | 🟢确认范围<br>🔴控制范围 | |
| **3.进度管理** | | 🔵规划进度管理<br>🔴定义活动<br>🔴排列活动顺序<br>🔴估算活动持续时间<br>🔴制定进度计划 | | 🔴控制进度 | |
| **4.成本管理** | | 🔵规划成本管理<br>🟢估算成本<br>🔵制定预算 | | 🔴控制成本 | |
| **5.质量管理** | | 🔵规划质量管理 | 🔴管理质量 | 🔴控制质量 | |
| **6.资源管理** | | 🔵规划资源管理<br>🟢估算活动资源 | 🟢获取资源<br>🔴建设团队<br>🔴管理团队 | 🔴控制资源 | |
| **7.沟通管理** | | 🟢规划沟通管理 | 🔴管理沟通 | 🔴监督沟通 | |
| **8.风险管理** | | 🔵规划风险管理<br>🔴识别风险<br>🔴定性风险分析<br>🔴定量风险分析<br>🔴规划风险应对 | 🔴实施风险应对 | 🔴监督风险 | |
| **9.采购管理** | | 🔵规划采购管理 | 🟢实施采购 | 🔴控制采购 | |

> **记忆口诀：** 启动组→识人章程 | 执行组→实施管理、获取建设 | 监控组→确认监控<br>
> **监控过程共性：** 子监控输入必有 ①项目管理计划 ②工作绩效数据；子监控输出必有 ①变更请求 ②工作绩效信息
